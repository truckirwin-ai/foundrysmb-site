# Contractor Bid Automation Starter Pack

**Price:** $99  
**Buyer:** contractor, remodeler, specialty trade, or services firm that quotes custom work  
**Promise:** Reduce estimate prep drag and create more consistent bid follow-up.

## Included assets

- PDF guide: "From Lead Notes to Bid Draft"
- Bid intake worksheet
- Estimate prep prompt pack
- Scope clarification email templates
- Follow-up sequence for open bids
- SOP for saving project context
- Demo script for a bid desk teardown

## Narrow problem solved

The pack helps teams move from scattered emails, photos, notes, and site details into a cleaner bid prep package. It does not replace estimator judgment.

## Demo script outline

1. Paste anonymized project notes into the intake prompt.
2. Generate missing-scope questions.
3. Draft a bid-prep checklist.
4. Draft the client clarification email.
5. Draft follow-up messages for day 2, day 7, and day 14.
