# 10-Day Founder-Led Sales Sprint

## Goal

Book paid AI Revenue Sprint calls with owner-led businesses that have visible workflow pain.

## Daily activity targets

- 20 new manual reaches per day.
- 5 follow-ups per day.
- 2 discovery calls booked per day target.
- Book the next call before ending every live call.

## ICP

- Owner-led business.
- 2-50 people.
- Visible manual workflow pain.
- Revenue depends on speed, follow-up, proposals, documentation, or service delivery consistency.
- Buyer can approve $1,500 without committee drag.

Priority verticals:

- Contractors and specialty trades.
- Professional services.
- Training and enablement businesses.
- Operations-heavy small teams.

## 10-day plan

### Day 1: Build the list

- Define 4 niches.
- Create 25 initial lead slots.
- Find owner/operator contact paths.
- Score each lead high, medium, or low.

Pass test: 25 leads are in the tracker with a reason for fit.

### Day 2: Send first 20

- Personalize based on visible workflow pain.
- Send template A or B.
- Log every touch.

Pass test: 20 first-touch messages sent.

### Day 3: Send next 20 and follow up warm replies

- Add 20 new leads.
- Reply same day to any response.
- Offer two call windows.

Pass test: at least 40 total first touches sent.

### Day 4: Publish proof content

- Post one workflow teardown.
- Send 20 messages referencing the teardown when relevant.
- Follow up day 2 sends.

Pass test: one public proof post and 60 total first touches.

### Day 5: Run calls and tighten pitch

- Run discovery calls.
- Score each call.
- Ask for the paid sprint when fit is 14+.

Pass test: every call has a next step, no vague "keep in touch."

### Day 6: Niche focus

- Identify which vertical generated replies.
- Build 40 more targets in that vertical.
- Send 20 focused messages.

Pass test: one vertical is named as current best bet.

### Day 7: Offer deadline

- Send follow-up with "two sprint slots next week."
- Publish before/after workflow post.

Pass test: every active lead has a dated next action.

### Day 8: Referral asks

- Ask warm contacts for one owner who has workflow chaos.
- Send 20 new messages.

Pass test: 5 referral asks sent.

### Day 9: Proposal day

- Convert fit calls into sprint proposals.
- Send payment links or written confirmations.

Pass test: every qualified lead receives a clear $1,500 ask.

### Day 10: Review and repeat

- Count touches, replies, calls, proposals, closes.
- Pick next niche.
- Improve one asset based on objections.

Pass test: next 10-day sprint starts with better targeting and copy.

## Metrics

- Outreach sent.
- Replies.
- Positive replies.
- Calls booked.
- Calls completed.
- Sprint proposals sent.
- Paid sprints closed.
- Install opportunities created.
